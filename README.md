# Tour and Travel Management System

A web-based Tour and Travel Management System developed using Java, JSP, Servlets, MySQL, HTML, CSS, and Bootstrap.

---

## Features

- User Registration
- Login System
- Ticket Booking
- Admin Dashboard
- Contact Form

---

## Technologies Used

- Java (JSP & Servlets)
- HTML
- CSS
- Bootstrap
- MySQL

---
